# my-study-log

考研备考用的个人学习与生活时间记录（纯前端，数据存在浏览器本地）。

在线演示：<https://dooorr.github.io/my-study-log/>

**技术栈：** HTML / CSS / JavaScript，无框架；`localStorage` 持久化；GitHub Pages 部署。

## 功能概览

### 首页看板

- 今日学习、连续打卡、**和自己比**的历史排名
- 倒数日、今日目标、科目总进度条
- 当前复习阶段摘要（若在阶段日期内）
- 四宫格入口：计时 / 今日 / 统计 / 设置

### 计时

- 普通正计时、番茄钟（默认 25+5 分钟，可改）
- 科目：数分、高代、英语、政治；生活：睡觉、玩手机；可自定义
- 子任务（如「4.2」），预设库 + 计时时一键点选
- 圆环进度、完成动效

### 今日

- 记睡眠、手动补录
- 今日记录列表（可拖 ⋮⋮ 调整显示顺序）
- 清空今日

### 统计

- **学习日排行榜**（只和自己历史比）+ 近 7 天 vs 上一周期
- **阶段统计**（基础 / 强化 / 冲刺等自定义阶段 + 科目分布）
- 科目复习进度、学习月历、7 天柱状图、30 天分布
- 时间占比饼图、子任务统计
- 导出 ICS（导入系统日历，非实时同步）

### 更多

- 复习阶段、科目进度目标、每日目标、番茄钟设置
- 深色模式、自定义类别、子任务预设
- JSON 导出 / 导入（支持拖文件到页面导入）
- 添加到主屏幕说明

### 手机

- 底栏：首页 / 计时 / 今日 / 统计 / 更多
- 建议 Safari 打开并「添加到主屏幕」

## 数据与备份

- 打卡数据**不会上传到 GitHub**，只存在本机浏览器
- 清缓存会丢数据，请定期「更多 → 导出 JSON」
- 电脑与手机**不自动同步**，换设备用导出 / 导入
- 备份 JSON 字段：`logs`、`customCategories`、`subtasksLibrary`、`dailyGoalHours`、`pomodoro`、`timerMode`、`dayLogOrder`、`countdown`、`subjectGoals`、`studyPhases`

## 本地运行

双击 `index.html`，或：

```bash
py -m http.server 8080
```

浏览器打开 `http://127.0.0.1:8080`。局域网访问见本地 `使用说明.txt`（`.bat` 等脚本不上传仓库）。
