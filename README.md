# my-study-log

考研备考用的个人学习与生活时间记录（纯前端，数据存在浏览器本地）。

在线演示：<https://dooorr.github.io/my-study-log/>

**技术栈：** HTML / CSS / JavaScript，无框架；`localStorage` 持久化；GitHub Pages 部署。

## 功能概览

### 首页看板

- 今日学习、连续打卡、历史学习日排名
- 倒数日、今日目标、科目总进度条
- 当前复习阶段摘要（若在阶段日期内）
- 功能入口：计时 / 今日 / 统计 / 时间轴 / 设置

### 计时

- 普通正计时、番茄钟（默认 25+5 分钟，可改）
- 科目：数分、高代、英语、政治；生活：睡觉、玩手机、吃饭、取快递、打电话；可自定义
- 子任务（如「4.2」），预设库 + 计时时一键点选
- **备注**（可选）：计时结束写入记录
- 圆环进度、完成动效；结束自动记录时段（供时间轴展示）

### 今日

手机端分三个子页，一页一块、不用长滑：

| 子页 | 内容 |
|------|------|
| **记录** | 今日概览 + 记录列表 |
| **日程** | 0:00–24:00 一日时间轴（竖向，色块高度对应时长） |
| **记一笔** | 记睡眠、手动补录 |

- 记录默认按**结束时间从新到旧**（早起先记的睡眠在下方）；可拖 ⋮⋮ 手动调序
- 每条记录可点「备注」编辑说明
- 记完睡眠/补录后自动切到「记录」页

### 统计

- 学习日排行榜 + 近 7 天 vs 上一周期
- **阶段统计**（基础 / 强化 / 冲刺等自定义阶段 + 科目分布）
- 科目复习进度、学习月历、7 天柱状图、30 天分布
- 时间占比饼图、子任务统计
- 导出 ICS（导入系统日历，非实时同步）

### 设置（右上角 ⚙）

- 主题切换：☀️ 🎨 🌊 🌲 🌙（设置页点选，或右上角 🌙 循环切换）
- 复习阶段、科目进度目标、每日目标、番茄钟设置
- 自定义类别、子任务预设
- JSON 导出 / 导入（支持拖文件到页面导入）
- 添加到主屏幕说明

### 手机

- 底栏四格：**首页 / 计时 / 今日 / 统计**（设置进右上角 ⚙）
- 建议 Safari 打开并「添加到主屏幕」

## 数据与备份

- 打卡数据**不会上传到 GitHub**，只存在本机浏览器
- 清缓存会丢数据，请定期「设置 → 导出 JSON」
- 电脑与手机**不自动同步**，换设备用导出 / 导入
- 单条记录含 `startAt` / `endAt`（有则进时间轴）；备份 JSON 主要字段：

  `logs`、`customCategories`、`subtasksLibrary`、`dailyGoalHours`、`pomodoroWorkMin`、`pomodoroBreakMin`、`timerMode`、`dayLogOrder`、`countdown`、`subjectGoals`、`studyPhases`

- 主题、当前 tab 等 UI 偏好存在 `localStorage`，**不含在 JSON 备份里**

## 本地运行

双击 `index.html`，或：

```bash
py -m http.server 8080
```

浏览器打开 `http://127.0.0.1:8080`。局域网访问见本地 `使用说明.txt`（`.bat` 等脚本不上传仓库）。
