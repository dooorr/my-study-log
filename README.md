# 考研学习打卡 · 数学学硕

纯前端小工具，**不用装 Node、不用服务器**。数据存在浏览器 `localStorage`。

## 打开方式

**电脑**：双击 `index.html`

**手机**：见 **`使用说明.txt`**（局域网 / 隧道 / GitHub Pages 等）

```powershell
cd "d:\Desktop\x\考研学习打卡"
start index.html
```

## 功能

- 学习：数分 / 高代 / 英语 / 政治；生活：睡觉 / 玩手机；可自定义类别
- 子任务：如「数分 · 4.2」，可预设、一键点选
- 专注计时、记睡眠、手动补录、今日概览、连续打卡
- 手机底栏：计时 / 今日 / 统计 / 更多
- 近 7 天学习图、30 天分布、子任务统计
- 导出 / 导入 JSON 备份

## 注意

- 清浏览器缓存会丢数据，建议每周导出一次备份
- 仅本机有效，不联网
